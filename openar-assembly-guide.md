# OpenAR 2.0 组装指南

> 详细步骤说明，从零开始组装
> 预计时间：4-8小时（第一次做）

---

## 阶段一：准备（1-2小时）

### 步骤1：检查零件
- [ ] 所有电子元件到货
- [ ] 3D打印件打印完成
- [ ] 光学元件（分光镜、透镜、镜子）准备就绪
- [ ] 工具齐全

### 步骤2：3D打印件处理
**目标：** 给所有需要螺纹的孔攻丝

**操作：**
1. 找到所有标有"Threads"的孔（根据README文件）
   - optics-frame: 12×M3, 1×M2
   - reflective-glass-holders: 各1×M3
   - electronic-display-adapter: 4×M3
   - TFT-adapter: 4×M3, 4×M2
   - headband-premade: 4×M3
   - CPU-holder: 2×M3

2. 攻丝步骤：
   ```
   a. 确认孔径（M2对应1.6mm底孔，M3对应2.5mm底孔）
   b. 丝锥垂直插入孔中
   c. 缓慢顺时针旋转，每转1/4圈回退一点（断屑）
   d. 直到旋转阻力突然变小，说明到底了
   e. 逆时针退出丝锥
   ```

3. **测试**：拧入对应螺丝，应顺畅不卡顿

**⚠️ 注意：** 3D打印件较软，不要用力过猛，以免滑丝

## 阶段二：电子电路制作（1-2小时）

### 步骤3：电压调节电路（3.3V）

**原理：**
电池(3.7V) → AMS1117-3.3 → 3.3V输出

**电路图：**
```
        10uF              100nF
电池+ ──┤├────┬───────┤├──────┬──→ 3.3V输出
            │   AMS   │
电池- ──────┴───1117──┴────────┴──→ GND
```

**操作：**
1. 在洞洞板上规划布局（约3cm×2cm区域）
2. 插入AMS1117-3.3（注意引脚方向：Vin-Vout-GND）
3. 焊接输入电容（10uF）在Vin和GND之间
4. 焊接输出电容（100nF）在Vout和GND之间
5. 焊接输入线（接电池+）
6. 焊接输出线（接3.3V设备）
7. 焊接GND线（公共地）
8. 检查焊点，确保无短路

**测试：**
1. 电池连接到输入
2. 万用表测量输出应为3.3V±0.1V

### 步骤4：焊接LCD屏幕排针

**操作：**
1. 将排针（2.54mm间距）插入LCD背面焊盘
2. 用斜口钳剪到合适长度
3. 电烙铁焊接每个引脚
4. 检查无短路

**引脚定义（ST7789）：**
- GND → 地
- VCC → 3.3V（注意：有些是5V tolerant，但建议3.3V）
- SCL/SCK → SPI时钟
- SDA/MOSI → SPI数据
- RES/RST → 复位
- DC → 数据/命令选择
- CS → 片选（可接地常选）
- BL/LED → 背光（接3.3V）

### 步骤5：连接Blue Pill和LCD

**接线表：**
| LCD引脚 | Blue Pill引脚 | 功能 |
|---------|--------------|------|
| GND | G | 地 |
| VCC | 3.3V | 电源 |
| SCL | PB3 | SPI1_SCK |
| SDA | PB5 | SPI1_MOSI |
| RES | PB4 | GPIO复位 |
| DC | PB6 | GPIO数据命令 |
| CS | GND | 常选通 |
| BL | 3.3V | 背光常亮 |

**操作：**
1. 剪适当长度的杜邦线（母对母）
2. 按上表连接
3. 用热熔胶或胶带整理线束

### 步骤6：连接电池和充电

**接线：**
```
电池+ ──┬──→ 充电模块B+
       └──→ 稳压电路输入

电池- ──┬──→ 充电模块B-
       └──→ 稳压电路GND

充电模块out+ ──→ 稳压输入
充电模块out- ──→ GND

稳压输出3.3V ──→ LCD VCC、Blue Pill 3.3V
稳压GND ───────→ LCD GND、Blue Pill G、充电模块-
```

**注意：**
- 电池必须有保护板（PCM）
- 充电模块的OUT直接给稳压芯片输入
- 所有地线(GND)连在一起

## 阶段三：光学组装（1.5-2小时）

### 步骤7：切割反射玻璃

**材料：** 分光镜（半透半反镜）50×50mm

**目标：** 切成两片，每片约 20×25mm

**操作：**
1. 用尺子和记号笔在分光镜上画线
2. 用玻璃刀沿画线用力划（听到"嘶嘶"声）
3. 在划痕下方垫细棍
4. 双手按住两侧，向下压，玻璃会沿划痕断裂
5. 用砂纸打磨边缘（400目→800目→1200目）

**⚠️ 安全：** 戴手套和护目镜，玻璃碎片危险

### 步骤8：准备透镜-反射镜组合

**材料：** 
- +3.5D老花镜镜片（拆下的）
- 小镜子（从化妆镜拆下）
- 胶水

**操作：**
1. 清洁透镜和镜子表面
2. 将镜子粘在透镜后方（形成一个组合体）
3. 确保镜子与透镜平行
4. 总厚度不超过 38×45mm（适配holder）

### 步骤9：组装光学框架

**顺序：**
1. **安装透镜-反射镜组合**
   - 将组合体放入 lens-mirror-holder
   - 用M3螺丝固定

2. **安装显示屏**
   - LCD装入 electronic-display-holder
   - 连接 electronic-display-adapter
   - 用M3螺丝固定到光学框架左侧

3. **安装反射玻璃支架**
   - 将杠杆(lever)装到框架上
   - 装上 glass-holders（左右各一）
   - 插入 glass-sockets
   - 装上 glass-grabbers

4. **插入反射玻璃**
   - 将切好的两片玻璃以45度角插入grabbers
   - 用胶水或摩擦固定

5. **安装调节机构**
   - 装上 lever-pusher-for-bolt
   - 装上 lever-pusher-for-spring
   - 装上 screw-pin 到M2螺栓上

6. **安装外壳**
   - 装上 frame-cover
   - 装上 optics-outer-shell

**调节瞳距：**
- 拧紧/松开M2螺栓，杠杆会推动反射玻璃左右移动
- 调节到适合你双眼的距离（一般58-72mm）

## 阶段四：头带组装（0.5-1小时）

### 步骤10：组装头带

**操作：**
1. 将 battery-holder 固定到头带上
2. 放入锂电池
3. 将 charger-circuit-holder 和 cover 装上
4. 固定 TP4056充电模块
5. 安装 CPU-circuit-holder，放入Blue Pill
6. 装上 voltage-distributor-circuit-holder，固定洞洞板电路
7. 装上 headband-cover-left 和 right

### 步骤11：连接光学和头带

**操作：**
1. 将 optics-to-headband-locking-piece 装到光学框架背面
2. 对准头带上的 optics-to-headband-socket
3. 用M3螺栓连接

**特点：**
- 这个连接允许你向上翻起光学部分，不摘下头带也能看外界

## 阶段五：软件烧录（0.5-1小时）

### 步骤12：安装开发环境

**需要：**
- Windows/Mac电脑
- STM32CubeIDE（免费）

**步骤：**
1. 下载安装 STM32CubeIDE
2. 打开项目文件夹中的 screenTest 工程

### 步骤13：烧录程序

**操作：**
1. 用ST-Link连接Blue Pill（4根线：SWDIO, SWCLK, GND, 3.3V）
2. 打开STM32CubeIDE
3. 点击锤子图标编译
4. 点击虫子图标调试/烧录
5. 等待完成

**测试：**
- 烧录成功后，LCD应该显示旋转立方体
- 如果黑屏，检查接线

---

## 阶段六：最终组装和调试（0.5-1小时）

### 步骤14：整理线缆

**操作：**
1. 将所有杜邦线束整理整齐
2. 用热熔胶或扎带固定
3. 确保不会拉扯到接头
4. 留出足够长度让光学部分能翻起

### 步骤15：最终测试

**检查清单：**
- [ ] 开机后LCD正常显示
- [ ] 能看到悬浮在眼前的立方体
- [ ] 双眼都能看到图像（立体视觉）
- [ ] 调节M2螺栓能改变图像位置
- [ ] 充电模块工作正常
- [ ] 电池续航至少1-2小时
- [ ] 佩戴舒适，头带不松动

### 步骤16：优化调整

**常见问题处理：**

**问题1：图像模糊**
- 调节透镜与眼睛的距离
- 清洁光学元件
- 确保反射玻璃角度准确45度

**问题2：只看到单眼图像**
- 调节瞳距（M2螺栓）
- 调整左右反射玻璃角度

**问题3：图像太暗**
- 检查LCD背光是否开启
- 清洁反射玻璃和透镜
- 检查电池电压

**问题4：重影/鬼影**
- 这是半透半反镜的正常现象
- 调暗环境光线
- 无法完全消除，只能减轻

## 故障排除FAQ

**Q: 屏幕完全不亮**
A: 检查1) 电池有电吗 2) 3.3V稳压输出正常吗 3) 背光引脚接了吗 4) LCD接线正确吗

**Q: 有背光但没图像**
A: 检查SPI接线（SCL/SDA），重新烧录程序

**Q: 图像有但模糊**
A: 调节透镜位置，清洁光学元件，检查眼睛到反射玻璃的距离

**Q: 头晕/恶心**
A: 这是VR/AR常见问题。休息一会儿，确保双眼图像对齐，降低使用时间。

**Q: 3D打印件断裂**
A: 检查打印填充率（建议30%以上），必要时重新打印更厚的部件。
