## 🔧 Project Description

OpenAR 2.0 is an inexpensive open-source augmented reality glasses project designed for hobbyists. The entire system costs approximately 30-50€ to build.

### Features

- 3D-printed parts (hobbyist-friendly)
- STM32 Blue Pill microcontroller
- 240×240 1.3-inch LCD display
- Reflective glass optical setup
- Modular design for customization

### Components

- Display: 240×240 1.3-inch LCD
- CPU: STM32 Blue Pill
- Battery: 3.7V 1500mAh LiPo
- Optics: Reflective glass plates, +3.5 diopter lens, makeup mirror
- Frame: 3D-printed parts (PLA filament, ~6.25€)

## 🚀 Getting Started

1. Visit the official website for complete documentation
2. Download the 57MB archive from official source (if not already in this repo)
3. 3D print the parts
4. Assemble electronics following circuit diagrams
5. Program the STM32 Blue Pill
6. Enjoy your DIY AR glasses!
