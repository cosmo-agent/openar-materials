## What to Extract

After downloading `OpenAR-2.0.zip` (57MB), extract the contents into this repository:

```
OpenAR-2.0/
├── docs/           # Documentation files
├── models/         # 3D models (.stl, .3mf, .f3d)
├── code/           # STM32 demo programs
├── circuit/        # Circuit diagrams
├── README.md       # This file
└── LICENSE.md      # License information
```
